- Все хранящиеся значения можно запросить в виде стандартного массива.

### ЗАДАНИЕ 1 (6) - Инкапсуляция в классе Line

```java
public class Line implements Cloneable {
    private Point start;
    private Point end;
    
    // Конструктор с защитой от общего доступа
    public Line(Point start, Point end) {
        if (start == null || end == null) {
            throw new IllegalArgumentException("Точки начала и конца не могут быть null");
        }
        if (start.equals(end)) {
            throw new IllegalArgumentException("Точки начала и конца не могут совпадать");
        }
        // Создаем копии точек для защиты инкапсуляции
        this.start = new Point(start.getX(), start.getY());
        this.end = new Point(end.getX(), end.getY());
    }
    
    // Геттеры возвращают копии для защиты от модификации
    public Point getStart() {
        return new Point(start.getX(), start.getY());
    }
    
    // Вычисление длины как целого числа
    public int getLength() {
        double dx = end.getX() - start.getX();
        double dy = end.getY() - start.getY();
        return (int) Math.sqrt(dx * dx + dy * dy);
    }
}
```

**Комментарии:**

- Конструкторы проверяют входные данные и создают копии объектов
- Геттеры возвращают копии для предотвращения внешнего изменения состояния
- Сеттеры также создают копии принимаемых точек
- Проверки исключений гарантируют корректность данных

    private final int[] values;
    
    public ImmutableList(int... values) {
        if (values == null) {
            throw new IllegalArgumentException("Массив не может быть null");
        }
        // Создаем копию для обеспечения неизменяемости
        this.values = Arrays.copyOf(values, values.length);
    }
    
    public ImmutableList set(int index, int value) {

        return new ImmutableList(newValues); // Возвращаем новый объект
    }
}
