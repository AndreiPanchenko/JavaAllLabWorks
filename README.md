## Лабораторная работа №6: Аннотации и тестирование

### Темы
- Создание пользовательских аннотаций
- Reflection API для обработки аннотаций
- JUnit 5 – модульное тестирование

### Реализованные аннотации
| Аннотация | Цель | Свойства |
|-----------|------|-----------|
| `@Invoke` | метод | – |
| `@Default` | тип, поле | `Class<?> value()` |
| `@ToString` | тип, поле | `Mode value()` (YES/NO, по умолч. YES) |
| `@Validate` | тип, аннотация | `Class<?>[] value()` |
| `@Two` | тип | `String first()`, `int second()` |
| `@Cache` | тип | `String[] value()` (по умолч. пустой массив) |

### Обработчики
- `AnnotationProcessor` – единый класс с методами `processInvoke`, `processDefault`, `processToString`, `processValidate`, `processTwo`, `processCache`

### Тестирование (JUnit)
- `ToStringTest` – проверка фильтрации полей (YES/NO)
- `CacheAndInvokeTest` – интеграционный тест совместной работы аннотаций

### Результаты тестирования
- Все тесты пройдены (100% успешность)
- Продемонстрирована работа Reflection: вызов методов по `@Invoke`, чтение параметров аннотаций
